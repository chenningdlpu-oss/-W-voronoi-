# -W-voronoi-
在进行非晶合金分析时通常需要复杂地计算W值及Voronoi多面体，对此，本项目设计了一款软件能够实现此类计算，并对于voronoi多面体前15个进行统计分析。有效地减少了该重复分析对科研造成的工作量。

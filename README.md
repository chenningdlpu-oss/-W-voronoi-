# Analysis software for local fifth symmetry and voronoi polyhedra of metallice glasses
在进行非晶合金分析时通常需要复杂地计算W值及Voronoi多面体，对此，本项目设计了一款软件能够实现此类计算，并对于voronoi多面体前15个进行统计分析。有效地减少了该重复分析对科研造成的工作量。
在运行前，需要利用ovito软件得到非晶合金的voronoi多面体数据，在准备好数据后使用该软件能够快速地分析非晶合金的平均局域五次对称性，并筛选符合voronoi多面体分析的原子，剔除不符合的原子进行voronoi多面体分析。绘制统计图如右图所示：
